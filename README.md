# Tanger 的知识库

## 简介

本仓库为 Tanger 的仓库，本知识库的模板为[Docusaurus](https://www.docusaurus.cn/)进行基本的架构，包管理为 yarn，前端框架为 react，本仓库在技术上的内容并不多，本仓库的主要内容是本仓库的最大的亮点 ⭐。
